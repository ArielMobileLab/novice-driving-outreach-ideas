# אתר איסוף רעיונות: ליווי נהגים צעירים

## מה כלול

- `index.html`: אתר מלא בעברית ובכיווניות RTL.
- `google_apps_script.gs`: שירות אופציונלי לשמירה משותפת ב-Google Sheets.
- ללא שירות שיתוף, האתר שומר את הנתונים רק בדפדפן הנוכחי.
- ניתן לייצא CSV או JSON ולייבא גיבוי JSON.
- לשונית הדירוג מחשבת ציון משוקלל לפי משקלי הקול הקורא: 20%, 20%, 20%, 20%, 5%, 15%.

## הפעלה מקומית

פתחו את `index.html` בדפדפן. אין צורך בהתקנה.

## פרסום האתר

אפשר להעלות את התיקייה לשירות אירוח סטטי, למשל GitHub Pages, Netlify או שרת האוניברסיטה.
מספיק להעלות את `index.html`.

## חיבור ל-Google Sheets

1. צרו גיליון Google Sheets חדש.
2. בתפריט בחרו Extensions ואז Apps Script.
3. מחקו את הקוד הקיים והדביקו את תוכן `google_apps_script.gs`.
4. בחרו Deploy, אחר כך New deployment, ואז Web app.
5. Execute as: Me.
6. Who has access: Anyone with the link.
7. אשרו הרשאות והעתיקו את כתובת ה-Web App.
8. פתחו את האתר, לחצו "הגדרות שיתוף" והדביקו את הכתובת.

## מגבלות

- אין מנגנון התחברות. יש לשתף את הקישור רק עם חברי הצוות.
- דירוגי הצוות נשמרים בדפדפן של כל מדרג ואינם נשלחים ל-Google Sheets.
- לשיתוף מאובטח יותר נדרש שרת עם אימות משתמשים.
